BashPass
                                                                 |
|   BashPass is a command-line based password manager written in Bash.       |
|   It uses GPG to encrypt/decrypt the files where the passwords are         |
|   stored. This means the passwords are 100% stored locally, so you don't   |
|   have to trust a third party to store your passwords. This is also one    |
|   of the projects I am most proud of.                                      |
|                                                                            |
|   Some features:                                                           |
|   ·	Add/edit/delete passwords                                            |
|   ·	Generate passwords                                                   |
|   ·	Clipboard support                                                    |
|   ·	Safely copy passwords between different devices                      |
|                                                                            |
|   Check out the documentation to get started with BashPass.                |
|                                                                            |

Gnoti
                                                                    |
|   Gnoti is a Gnome Shell extension, to change the position and duration    |
|   of the notification banner.                                              |
|                                                                            |
|   As is the case with all Gnome Shell extensions, it is written in         |
|   JavaScript.                                                              |
|                                                                            |
|   Gnoti should be with Gnome version 3.36 and above. To be more specific   |
|   this includes the following versions:                                    |
|   ·	3.36                                                                 |
|   ·	3.38                                                                 |
|   ·	40                                                                   |
|   ·	41                                                                   |
|   ·	42                                                                   |
|                                                                            |
|   The installation of the extension is pretty easy. You just run the       |
|   installer (which is a shell script) and it will install the extension    |
|   for you. The only thing you need to do is to specify a position as a     |
|   argument to the installer.                                               |
|                                                                            |
|   The possible positions are:                                              |
|   ·	top-left                                                             |
|   ·	top-center                                                           |
|   ·	top-right                                                            |
|                                                                            |
|   Check out the repository for more information.                           |
|                                                                            |
