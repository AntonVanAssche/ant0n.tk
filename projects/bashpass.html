BashPass
                                                                 |
|   BashPass is a command-line based password manager written in Bash.       |
|   It uses GPG to encrypt/decrypt the files where the passwords are         |
|   stored. This means the passwords are 100% stored locally, so you don't   |
|   have to trust a third party to store your passwords.                     |
|                                                                            |
|   Although this isn't a complex program, it is one of the projects I'm     |
|   most proud of. Mainly because I've learned a lot about Bash and how      |
|   to use it. For example, I've tried to use as much Bash built-ins as      |
|   possible, such as `read` to replace the `sleep` command (`sleep` isn't   |
|   a built-in). I've also written a few functions to replace programs such  |
|   as `cut` and `grep`, to make it more efficient. While the speed          |
|   improvements are minor (almost negligible), it helps to make the code    |
|   more compatible with other operating systems.                            |
|                                                                            |
|   Dependencies:                                                            |
|   ·	Bash >= 3.0                                                          |
|   ·	gpg or gpg2      (Encrypt/Decrypt)                                   |
|   ·	xclip            (Clipboard support)                                 |
|   ·	rsync            (Synchronization support)                           |
|                                                                            |
|   While `xclip` and `rsync` aren't required to run the program if you      |
|   aren't using functionalities such as `--copy` and `--sync`, they are     |
|   recommended in case you need them. And after all, they can come in       |
|   handy for a number of other tasks on your system (creating backups,      |
|   etc.).                                                                   |
|                                                                            |
|   Some features:                                                           |
|   ·	Add/edit/delete passwords                                            |
|   ·	Generate passwords                                                   |
|   ·	Clipboard support                                                    |
|   ·	Safely copy passwords between different devices                      |
|                                                                            |
|   Check out the documentation to get started with BashPass.                |
|                                                                            |
