Raspberry Pi NAS

This project is basically the same as my Raspberry Pi Home Server project the only main difference is that this will setup my other Raspberry Pi 4 as a Network Attached Storage server using SATA HHD's. This means you'll need a special hat board (Raspberry Pi - Quad NAS Kit, 4x SATA) for your Raspberry Pi in order to use this project.

In this GitHub repository you will find a bunch of scripts to automate the setup process, but also a few configuration files and Grafana dashboards. It also includes a service file to mount the RAID 5 array on boot, this is needed because you are not able to mount the HDD's the '/etc/fstab' file. This is due to the fact that a driver has to be running to start the SATA hat.

As I've mentioned earlier, a RAID 5 array will be configured so you won't lose any data as long as only one of the HDD's fails or is removed.

Check out the README.md file for more information.
